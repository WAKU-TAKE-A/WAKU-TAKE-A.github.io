平滑化の1つ。「Bilateral Filtering for Gray and Color Images」に詳細な説明があります。

他の平滑化と比べて処理速度は遅いです。

フィルタを強くかけると（フィルタリングの直径up、座標空間 でのシグマup）、「漫画っぽく」べたっとした感じの画像になります。

diameter
フィルタリングの直径。正でない場合はsigmaSpaceから計算されます。「5 < d」で処理は遅くなります。リアルタイムな処理を行いたい時はd=5、オフラインの処理で十分ノイズを消したい時はd=9を試してみてください。
sigmaColor
色空間でのシグマ
sigmaSpace
座標空間でのシグマ
borderType:
フィルタリングに使用する画像外側の外挿方法

サンプル

サンプルマクロを用意しました。以下の手順で試してください。

  • こちらからzipファイルをダウンロードしてください。
  • 全て展開します。
  • BilateralFilter_Sample.ijmとlena.jpg、「File -> Open」で開きます。
  • BilateralFilter_Sample.ijmを、「Macros -> Run Macro」で実行してください。
