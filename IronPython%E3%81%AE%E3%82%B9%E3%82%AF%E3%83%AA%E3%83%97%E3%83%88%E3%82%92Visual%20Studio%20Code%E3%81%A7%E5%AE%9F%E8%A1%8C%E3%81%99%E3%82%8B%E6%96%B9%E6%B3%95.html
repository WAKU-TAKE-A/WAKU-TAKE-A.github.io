IronPythonのスクリプトをVisual Studio Codeで実行する方法

13th December 2019 at 8:52pm
IronPython

IronPythonのスクリプトファイルをVisual Studio Codeで実行・デバッグする方法です。2019年12月版です。

(1) 拡張機能「Python プラグイン」を追加します

Visual Studio Code 1.40.2 ＋ 拡張機能「Python プラグイン」の最新ではIronPythonは正常動作しません。しかし、指定したバージョンで拡張機能をインストールできますので、IronPythonが正常動作するバージョンに変更します。

まず、最新の「Python プラグイン」を追加します。

プラグインを右クリックすると以下が表示されます。「別のバージョンをインストール」を実行します。

「2018.7.1」を選択します。以降のバージョンでは正常動作しませんでした。

自動アップデートをオフにしておいてください。ファイル ⇒ 基本設定 ⇒ 設定で以下を設定してください。

念のため、こちらも。

(2) 「.vscod」フォルダの「settings.json」の「python.pythonPath」に適切なパスを設定します

「.vscod」フォルダが無い場合には、まず「フォルダを開く」で実行したいファイルのフォルダを開いてください。次に、ファイル ⇒ 基本設定 ⇒ 設定 ⇒ ワークスペースの設定を選択 の手順を行うことで作成されます。

{
    "python.linting.pylintEnabled": false,
    "python.pythonPath": "C:/IronPython27/ipy.exe"
}

インストールしたフォルダに設定してください。

(3) デバッグ ビューにおいて、歯車マークをクリックして構成を追加します

「Python プラグイン」のバージョンを変更した場合には、すでにある「launch.json」は削除して、再度構成を追加してください。

(4) デバッグ ビューにおいて、「Current File」か「Terminal (integrated)」あたりで実行してみてください

IronPythonを使うかCPythonを使うかでバージョンを切り替えるのが、吉かもしれません。

あと、何をしてもうまくいかない時は、Visual Studio Codeを完全アンインストールしてから、再インストールするとうまくいく場合があります。

完全アンインストールの方法は以下の通りです。

  1. Visual Studio Codeのアンインストール
  2. C:¥Users¥ユーザー名¥AppData¥Roaming¥Codeの削除
  3. C:¥Users¥ユーザー名¥.vscodeの削除
