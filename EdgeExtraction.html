画像の勾配からエッジを抽出するOpenCVの関数を使ったプラグインです。

1次微分フィルター

まず、こちらで画像の勾配に関する基礎的内容を読んでください。

記事の真ん中あたりの例が分かりやすいと思います。

x-1対象の画素（x）x+1
10200210

200のx方向の勾配の計算は、以下のようになります。

{ f(x+1) - f(x-1) } / 2 = ( 210 - 10 ) / 2 = 100

このような計算を応用して1次微分フィルターが作成されます。こちらの内容を読んでください。

様々な考え方で畳み込み用のカーネルを作成しています。

今回参考にしたホームページは「TheAILearner」です。画像処理のページはとても勉強になります。

【OCV_Sobel】

Sobelオペレーターによるエッジ検出を行います。

Sobelオペレーターは、ガウス平滑化（1D）と1次微分を組合せたものです。

【OCV_Scharr】

Scharrオペレーターによるエッジ検出を行います。

2次微分フィルター

【OCV_Laplacian】

ラプラシアンオペレーターによるエッジ検出を行います。

更に複雑な手法

【OCV_Canny】

「Canny Edge Detector」はエッジ検出の最も一般的なアルゴリズムの一つです。

  • ガウシアンフィルターを使った平滑化。
  • Sobelオペレーターを使ったエッジ抽出。
  • エッジの細線化。
  • ヒステリシスしきい値処理。

などの処理を組み合わせたアルゴリズムです。
