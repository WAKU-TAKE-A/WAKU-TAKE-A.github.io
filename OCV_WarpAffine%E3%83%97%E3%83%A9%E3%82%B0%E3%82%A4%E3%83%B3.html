簡単な説明

計算で導出した、もしくはファイルから読み込んだ2×3行列でアフィン変換します。

dst(x,y) = src(M11x + M12y + M13, M21x + M22y + M23)

関連するプラグインとしては、

OCV_GetAffineTransform
3組の対応する点からアフィン変換を計算します。
OCV_GetRotationMatrix2D
2Dの回転のアフィン行列を計算します。
OCV_InvertAffineTransform
アフィン変換を反転します。

サンプル

サンプルマクロを用意しました。以下の手順で試してください。

  • こちらからzipファイルをダウンロードしてください。
  • 全て展開します。
  • src.pngとRoiSet.zipとAffine_Macro.txt、「File -> Open」で開きます。
  • OCV__LoadLibraryを実行します。
  • Affine_Macro.txtを、「Macros -> Run Macro」で実行してください。

RoiManagerに入っている2つのROIは、元画像の3点と、アフィン変換後の対応する3点の位置データです。

OCV_WarpAffineやOCV_InvertAffineTransformは、ResultsTableのデータを文字列として読み込み数値に変換します。
その際に、"や'がある場合は無視して取り込みます。
